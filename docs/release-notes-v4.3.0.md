# 新機能追加・改善、バグ修正

・ホーム画面の表示を自由に編集できる機能を追加
→ プロフィール、学習実績、課題リストなどの表示・非表示と並び順を変更できるようになりました。課題の提出日を確認できるカレンダーや、非表示にした課題の復元・完全削除にも対応しました

・「テーマカラー変更」と「カスタム画像テーマ」を追加
→ サイト上の要素を選んで文字色・背景色・枠線色を変更できるほか、好きな画像をKu-LMSの背景に設定し、位置・拡大率・透過度を調整できるようになりました

・TOTP認証の設定をサポート
→ TOTPの設定方法をまとめた案内ページを追加し、大学の認証画面に表示されたQRコードからKLPFの自動ログイン用秘密鍵を設定できるようになりました

・「課題リストアップ」と「ホーム出席表示」の通信処理を改善
→ ホーム画面を開いた直後に課題や授業カードを押すと、システムエラーや別の授業へ遷移することがある不具合を修正しました

・「履修中科目のみ表示」を改善
→ 現在の学期・クォーターに合う科目だけを自動表示し、現在時刻と関係のない過去・未来の科目まで赤く表示される不具合を修正しました。自動表示中の検索ボタンにはチェック解除の案内を表示します

・ホーム画面と各種設定画面の表示・操作性を改善
→ 別URLから戻ったホーム画面への対応、24:00期限の課題日付、短い画面でのカレンダー表示、ポップアップ中のスクロール、テーマ設定の復元などを改善しました。「すべての機能を OFF」にすると追加メニューとテーマ適用も一時停止し、解除時に復元します
