Features

あなたの学習を加速する
9つの機能

日常的な操作を自動化し、学習により集中できる環境を提供します

🔐

自動ログイン

Ku-LMSやku-portへの面倒なログイン作業を完全自動化。設定した認証情報で瞬時にアクセスできます。

⏰

自動ログアウト無効化

作業中の意図しないログアウトを防止。長時間の学習セッションも安心して続けられます。

📝

課題リストアップ

散らばった課題情報を一箇所に集約。期限順に整理され、メール通知で見逃しを防げます。

🎥

Meetミュート参加

Google Meetに自動的にミュート状態で参加。うっかりマイクがオンになる心配がありません。

📚

履修科目フィルター

履修中の科目のみを表示する設定を記憶。毎回の絞り込み操作が不要になります。

🕐

授業時間表示

現在時刻と次の授業までの時間をリアルタイム表示。スケジュール管理がより簡単に。

🌙

ダークモード

目に優しいダークテーマでKu-LMSを表示。長時間の利用でも疲労を軽減します。

✅

自動出席

指定時間に自動で出席処理とMeet参加。朝の授業も寝坊を心配する必要がありません。

📖

教材一括開封

教材ページから詳細ページを開くことなく、選択した教材をワンクリックですべて開き、ステータスを参照済みにします。

💡 使い方

  • 設定の開き方: Chromeの画面で右クリックし、「[KLPF] 設定を開く」を選択します。
  • 初期設定: 推奨機能が導入時にオンになっています。
  • 自動ログイン: ご利用には統合認証IDとパスワードの入力が必要です。
    パスワードをブラウザに保存している場合はchrome://password-manager/passwords/kogakuin.ac.jpか
    chrome://password-manager/passwords/secioss.comにアクセスすることで素早くIDとパスワードを入手できます。

⚠️ インストール時の注意点

  • フォルダの選択: ZIPファイルではなく、解凍後のフォルダを選択してください。
  • エラーが出た場合: manifest.jsonファイルが含まれるフォルダを選択しているか確認してください。
  • フォルダの保管: 導入後、解凍したフォルダは削除しないでください。削除すると拡張機能が動作しなくなります。
